# Customer Support Insights & RCA Analytics (Mock Project)

Files included:
- data_cleaning.py: cleans mock CSVs and outputs cleaned files.
- analysis.py: computes KPIs and exports CX_Insights.xlsx.
- rca_pareto.py: Pareto analysis and 5-Whys template.
- Original mock CSVs: support_tickets.csv, call_logs.csv, feedback.csv (created earlier).

How to run:
1. python data/data_cleaning.py
2. python data/analysis.py
3. python data/rca_pareto.py

you can access the streamlit apllication here = https://customersupport-analytics.streamlit.app/
Outputs will be written to data and can be downloaded from the UI.
